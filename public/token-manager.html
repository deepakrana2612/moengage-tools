🔐 Token Manager

Copy your Bearer token from the browser DevTools Network tab and paste it here.
The server stores it in memory and injects it into all Flow API proxy calls.

How to grab the token from DevTools

  1. Open the Flow UI and log in
  2. Press F12 → Network tab → trigger any action in the UI
  3. Click any API request in the list
  4. Go to Headers → find Authorization: Bearer <token>
  5. Copy everything after Bearer and paste below
